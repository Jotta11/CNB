A — Dois botões no topo (toggle)

Mantém o layout atual de abas, mas adiciona um seletor de módulo no header antes das abas. Ao alternar, as abas mudam para o módulo selecionado.

C — Tela de seleção de módulo

Ao entrar no admin, exibe uma tela de boas-vindas com os dois módulos como cards grandes. Cada módulo tem sua própria navegação interna independente.
